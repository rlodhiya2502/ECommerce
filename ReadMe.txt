This repository is a demo solution to showcase the Microservice architecture and its implementation with Entity Framework (Repository) Patterns.

Problem Statement:

Acme industry is an ecommerce company trading in computer peripherals. They currently have a monolith application and looking for a solution to implement basic search functionality for the orders to use internal purposes. 

They have awarded you for writing a simple proof of concept solution explaining stakeholders to adopt the microservice architecture and its implementations.   

The core requirement is to search orders by customer id from the given schema.
